# scap

## supported platform
* macos
* linux

tested only neovim in both platform.


## install

```
Plug 'git@github.com:satoyan/scap.git'
```

## How to use

1. select lines which you want as visual mode.
2. press keys 'gs'
3. then web browser will open https://ray.so
